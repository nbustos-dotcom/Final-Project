# Final-Project
Text adventure game for final project

Player & Inventory Management (Person 1)
Implement Player class
Track current room, inventory, and actions like GET, DROP, INVENTORY
Handle interactions with items (knife, pliers, flashlight, key, letter)


Room & Map System (Person 2)
Implement Room class
Set up rooms, descriptions, connections (doors, left/right rooms, colors)
Place items in rooms
Handle LOOK command


Game Logic & Commands (Person 3)
Implement Game class (TextAdventure main class)
Handle commands: GO, SAVE, RESTORE, QUIT, GET, DROP, LOOK, INVENTORY
Track win/lose conditions (bomb, monster, escape)
Trigger puzzles and stationary monster


Puzzles, Obstacles, and Extra Mechanics (Person 4)
Implement puzzles: code, trivia, bomb wires, monster logic
Track which items are needed for each room (knife, pliers, key)
Add extra features if possible (stationary monster, points, score, branching endings)
